# 迭代路线

## 第一阶段：可联调版本 ✅ 已完成

- FastAPI 后端连接 MySQL
- TDesign Vue 管理端支持商品、订单、快递管理
- TDesign 小程序支持商品列表、购物车、下单、订单查询
- 订单创建和取消具备库存事务

## 第一阶段增强 ✅ 已完成

- 后端全面安全加固（速率限制、输入验证、LIKE 注入防护）
- 后端分页支持、订单状态流转校验、操作日志
- 后端管理员 CRUD、密码修改、商品 PATCH/DELETE
- 管理端响应式布局、侧边栏折叠、搜索筛选分页
- 管理端确认对话框、错误处理、管理员管理页面
- 小程序商品搜索、商品详情页、购物车角标
- 小程序取消订单、手机号验证、移除确认

## 第二阶段：可运营版本

### Phase 2A（核心功能）
- 接入微信登录，后端保存用户 `openid`
- 增加收货地址簿，支持一个用户维护多个收货地址
- 订单状态变化发送订阅消息

### Phase 2B（管理增强）
- 商品图片上传到对象存储
- 管理端操作日志
- 营收统计趋势图、库存预警、热销排行

## 第三阶段：上线增强

- 后端部署到 HTTPS 域名
- 接入微信支付
- 配送路线汇总和批量操作
- 快递扫码登记
- MySQL 自动备份和迁移脚本
- 管理员角色权限细化
