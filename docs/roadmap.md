# 迭代路线

## 第一阶段：可联调版本

- FastAPI 后端连接 MySQL。
- TDesign Vue 管理端支持商品、订单、快递管理。
- TDesign 小程序支持商品列表、购物车、下单、订单查询。
- 订单创建和取消具备库存事务。

## 第二阶段：可运营版本

- 接入微信登录，后端保存用户 `openid`。
- 增加老人地址簿，支持一个用户维护多个老人地址。
- 商品图片上传到对象存储。
- 接入微信支付或线下付款确认流程。
- 订单状态变化发送订阅消息。
- 管理端增加订单搜索、配送筛选、操作日志。

## 第三阶段：上线增强

- 后端部署到 HTTPS 域名。
- MySQL 自动备份和迁移脚本。
- 管理员角色权限。
- 快递扫码登记和批量导入。
- 营收统计、库存预警、配送路线汇总。
